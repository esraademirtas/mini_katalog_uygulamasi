## ✨ Özellikler
* **Modern Arayüz**: Apple stilinden ilham alan minimalist tasarım.
* **Akıllı Listeleme**: GridView yapısı ile ürün sergileme.
* **Navigasyon**: Sayfalar arası veri aktarımı ve Hero animasyonları. 
* **Sepet Simülasyonu**: Gerçek zamanlı tutar hesaplama. 

## 🛠️ Teknik Detaylar
* **Kullanılan Dil**: Dart 
* **Flutter Sürümü**: 3.x.x (Lütfen `flutter --version` ile kontrol ediniz) 
* **Klasör Yapısı**: Temiz kod prensiplerine uygun mimari. 

## 🚀 Çalıştırma Adımları 
1. Projeyi klonlayın: `git clone <repo-url>`
2. Paketleri yükleyin: `flutter pub get`
3. Uygulamayı başlatın: `flutter run`
